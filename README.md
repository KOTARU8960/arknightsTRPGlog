作成日：2025/2/8
最終更新：2025/2/9
## これは何？
ccfoliaで行ったアークナイツTRPGセッションのクリテイカルとエラー回数の統計を取るサイトです。  
~~現在はADでのダイスロールにのみ対応しています。~~  
2025/2/9のアプデでABロールにも対応しました。  
## 使い方
ccfoliaから出力したログを選択もしくはドラッグアンドドロップしてください。
